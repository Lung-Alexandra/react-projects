# React Projects

A collection of React and Next.js projects built while learning core frontend concepts, routing, data fetching, and deployment.

## Live Apps

### react-app2
- Live: https://react-app2-gx3o.onrender.com
- Covers React fundamentals: component composition, prop passing, basic state-driven UI, conditional rendering, and modal/backdrop patterns for user interactions.

### react-app3
- Live: https://react-app3-si42.onrender.com
- Covers client-side routing with `react-router-dom`, form handling, data fetching, CRUD-style integration with Appwrite TablesDB, environment-based configuration, and shared state via React Context for favorites management.

### react-app4
- Live: https://allreact-app4.onrender.com
- Covers Next.js App Router concepts: file-based routing, nested and dynamic routes, server components, loading and error boundaries, data fetching on the server, forms for creating meals, and cache/revalidation-related patterns.

### react-post-app
- Live: https://react-post-app-ab2t.onrender.com
- Covers SPA routing with React Router, route loaders/actions, form submission through router actions, API integration with `fetch`, separation of frontend and backend services, and deployment of a Vite frontend plus Express API.

## Other Project

### react-news-app
- Live: https://react-news-app-p05x.onrender.com
- Covers advanced Next.js App Router patterns such as nested layouts, parallel routes, catch-all segments, dynamic route handling, and page composition for multi-section content flows.
